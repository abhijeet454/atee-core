# ATEE Web

Next.js frontend for [ATEE Core](../README.md).

## Setup

```bash
npm install
cp .env.example .env.local
npm run dev
```

Configure `NEXT_PUBLIC_API_URL` and `NEXT_PUBLIC_WS_URL` in `.env.local` if the backend is not on `localhost:8000`.

See the [root README](../README.md) for full project documentation.
